# Hansei Portal

한세사이버보안고등학교의 시간표와 급식 페이지로 이동하는 정적 바로가기 페이지입니다.

## 주요 기능

- 시간표 페이지와 급식 페이지 빠른 이동
- 모바일 대응 레이아웃
- favicon, web manifest
- 전용 Python 정적 서버
- 보안 헤더와 허용 파일 목록 기반 서빙

## 실행

```bash
python3 hansei_server.py
```

기본 주소는 `http://127.0.0.1:3011`입니다. 필요하면 환경 변수로 바꿀 수 있습니다.

```bash
HOST=127.0.0.1 PORT=3011 python3 hansei_server.py
```

## 저장소 관리

배포 설정, 서버별 경로, 내부 주소, 인증서 경로, 로컬 요구사항 문서는 공개 저장소에 커밋하지 않습니다. 해당 파일은 `.gitignore`에서 제외합니다.
